前言

Apache Shiro 是 Java 的一个安全框架。功能强大，使用简单的Java安全框架，它为开发人员提供一个直观而全面的认证，授权，加密及会话管理的解决方案。

功能介绍

资源-角色-权限
登录认证，密码加密（Authentication, Authorization, Cryptography）
用户角色和权限放入缓存（Caching）
会话管理（Session Management）
